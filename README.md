# 🚀 Simulador de Produção Industrial

Aplicativo Android desenvolvido com foco em **simulação de processos produtivos**, utilizando conceitos de **matemática aplicada** para análise de desempenho, tempo e custo operacional.

---

## 📱 Sobre o Projeto

Este aplicativo permite simular diferentes cenários de produção industrial, onde o usuário pode inserir variáveis reais do processo produtivo e obter análises detalhadas como:

* Capacidade produtiva diária
* Comparação entre linhas de produção
* Tempo necessário para conclusão
* Estimativa de custo operacional

O projeto foi desenvolvido com o objetivo de aplicar conceitos matemáticos em um contexto prático, aproximando a teoria da realidade industrial.

---

## 🧠 Modelo Matemático

O sistema é baseado nas seguintes relações:

```text
Tempo disponível = H × 60

Produção = (Tempo × Eficiência × Operadores) ÷ Tempo por peça

Dias necessários = Q ÷ Produção

Custo total = Dias × H × C
```

Esses cálculos permitem transformar dados de entrada em informações estratégicas para tomada de decisão.
